성능테스트 서비스 비교

21 September 2017

서비스 기능 특징 비고
RedLine13 * AWS(아마존) 기반 성능테스트 서비스 제공
* AWS IAM 계정을 이용한 Agent 설치
 * 스팟 인스턴스 활용을 통한 비용 절감
* JMeter, Gatling, Python, PHP 및 NodeJS 지원
Loader.io * 클라우드 기반 부하 테스트 서비스.
* 10만 동접 테스트 가능.
 * 무료로 1분간 2URL 의 만명 동접 테스트 가능
* 테스트 리플레이(되돌려보기) 기능 제공
BlazeMeter * 서비스 및 내부설치로 테스트 플랫폼을 제공.
* 모바일 앱, 웹사이트 API 테스트 제공.
* 백만이상 동접 테스트 가능.

 * 무료로 50명 동접까지 10번 테스트 가능
* JMeter, Selenium 같은 오픈소스 호환됨
* 로드러너 스크립트를 JMeter로 트랜스폼 제공
HPE StormRunner * AWS 기반 웹 및 모바일 테스트 서비스 제공.
* 몇 분만에 설정되어 수백만 동접 테스트 가능.
 * 무료로 1개월간 100명 동접까지 10번 테스트 가능
* JMeter와 로드러너 스크립트 활용 가능.
* 다양한 도구 호환(New Relic, Dynatrace, WebPageTest,…).
